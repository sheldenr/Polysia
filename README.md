# 🧠 Polysia — The Adaptive Mandarin AI Tutor

Polysia is an adaptive Mandarin learning platform built to make fluency achievable.  
It blends intelligent spaced repetition, visual recall, and contextual AI conversation — creating a personalized path to fluency for every learner.

---

## 🚀 Overview

Polysia helps learners master Mandarin through three core learning modes:

1. **Study Deck** – Adaptive flashcards that mix review and new words dynamically (like Anki, but powered by AI).
2. **Guess the Picture** – Visual recall mode that strengthens character-to-meaning associations.
3. **Chat Mode** – Context-aware Mandarin conversation practice that adjusts tone, vocabulary, and difficulty.

The app tracks user performance across tones, grammar, and recall, using an internal **Adaptive Learner Model (ALM)** to optimize what appears next.
